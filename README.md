# growi-plugin-tree-builder

GROWI の記事ツリーを**カードで表現したツリー**として編集するスクリプトプラグインです。テンプレートをカードで選んで追加し、名前付け・階層配置でツリーを組み、ローカルに保存できます。

## 機能

- **ツリーのカード表示**: 各ノードを 1 枚のカードで表示。インデントと左ボーダーで階層を表現。
- **テンプレート**: 右パネルにテンプレートカードを表示。クリックで選択ノードの子（またはルート直下）に追加。ドラッグで任意の位置に追加。
- **編集**: タイトル編集、インデント / アウトデント、ドラッグで移動、削除。子の開閉。
- **GROWI から読み込み**: 現在パスの直下の子ページを取得し、ツリーとして表示。
- **保存**: ツリーをブラウザの localStorage に保存（ルートパスごと）。

## 必要な環境

- GROWI（オンプレ / GROWI.cloud）
- ブラウザから `/_api/v3/pages/list` が同一オリジンで利用できること

## インストール

1. GROWI の**管理画面** → **プラグイン** を開く
2. **スクリプトプラグイン** の「プラグインの URL」にこのリポジトリの URL を指定  
   - 例: `https://github.com/muro66/growi-plugin-tree-builder`
3. **インストール** → **有効にする**
4. サイドバーに **「📋 ツリー構築」** ボタンが表示されます。

**サイドバーにボタンが出ない場合**

- **画面左端**（画面のふち付近）に **「📋 ツリー構築」** のフォールバックボタンが出ていないか確認してください。クリックで同じパネルが開きます。
- ページを**再読み込み**（F5 または Ctrl+R）してから再度試してください。サイドバーは読み込み後に表示されるため、数秒後にボタンが現れることがあります。
- 別のページ（例: トップや任意の wiki ページ）を開いた状態で試してください。

## 使い方

### パネルを開く

- サイドバーの **「📋 ツリー構築」** をクリックすると、ツリー構築用のパネルが開きます。
- **ルートパス** は、パネルを開いた時点で表示している GROWI のページのパスです。別のページをルートにしたい場合は、そのページを開いた状態でパネルを開いてください。

### 画面の構成

| エリア | 内容 |
|--------|------|
| **左** | ルートパス表示、「GROWI から読み込み」「ツリーを保存（ローカル）」ボタン |
| **中央** | ツリー編集エリア（カードの並びで階層を表示・編集） |
| **右** | テンプレート一覧（セクション・概要・詳細・参照 など） |

---

### ツリーの編集（中央エリア）

#### カードの操作

| 操作 | やり方 |
|------|--------|
| **選択** | カードをクリックすると選択状態になります。テンプレートを「クリックで追加」したとき、選択中のカードの**子**として追加されます。 |
| **タイトル変更** | カード内のタイトル部分をクリックして文字を編集します。 |
| **インデント（子にする）** | カード右側の **→** ボタンをクリックすると、そのカードがひとつ上のカードの**子**になります。 |
| **アウトデント（親の兄弟にする）** | カード右側の **←** ボタンをクリックすると、そのカードが**親の兄弟**（ひとつ上の階層）に上がります。 |
| **削除** | カード右側の **×** ボタンをクリックすると、そのカードとその**子孫**がまとめて削除されます。 |
| **開閉** | 子を持つカードの **▼** をクリックすると、子の表示／非表示を切り替えられます。 |

#### ドラッグ＆ドロップ

- カード左端の **⋮⋮** をつかんでドラッグし、別の位置にドロップできます。
- **カードの上半分**にドロップ → そのカードの**直前の兄弟**として挿入
- **カードの下半分**にドロップ → そのカードの**直後の兄弟**として挿入
- **「ここにドロップで子として追加」** にドロップ → そのカードの**子の先頭**として追加
- **「ここにドロップで末尾に追加」** にドロップ → ルート直下の**末尾**に追加

---

### テンプレートの追加（右エリア）

- **クリック**  
  テンプレートカードをクリックすると、**選択中のカードの子**（何も選択していない場合は**ルート直下**）に、そのテンプレートの新規カードが 1 枚追加されます。
- **ドラッグ**  
  テンプレートカードをドラッグして、中央のツリーエリアの任意のドロップゾーンにドロップすると、その位置に新規カードが追加されます。

初期で用意されているテンプレート: **セクション** / **概要** / **詳細** / **参照**（いずれもタイトルは編集可能です）。

---

### GROWI から読み込み

- 左の **「GROWI から読み込み」** をクリックすると、現在のルートパスの**直下の子ページ**を GROWI の API で取得し、ツリーとして表示します。
- 既存の記事構成をツリー上で確認・編集したいときに使います。

---

### ツリーの保存（ローカル）

- 編集したツリーは**ルートパスごと**にブラウザの **localStorage** に自動で保存されます。
- 左の **「ツリーを保存（ローカル）」** をクリックすると、明示的に保存し、「ローカルに保存しました」と表示されます。
- 同じルートパスのページで再度パネルを開くと、前回のツリーが復元されます。
- ※ 保存先はブラウザ内のみです。別の端末やブラウザでは共有されません。

---

### 閉じる

- パネル右上の **「閉じる」** でパネルを閉じます。ツリーは自動保存されているため、次に開いたときに再表示されます。

---

## 開発

```bash
npm install
npm run build
```

## 計画

- Phase 1–2: カード表示・選択・編集・ドラッグ（実装済み）
- Phase 3: テンプレートパネル（実装済み）
- Phase 4: GROWI へのツリー保存（ページ API 連携は要検討、現状はローカル保存）
- Phase 5: コネクタ・キーボード・アクセシビリティ
