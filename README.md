# growi-plugin-tree-builder

GROWI の記事ツリーを**カードで表現したツリー**として編集するスクリプトプラグインです。テンプレートをカードで選んで追加し、名前付け・階層配置でツリーを組み、ローカルに保存できます。

## 機能

- **ツリーのカード表示**: 各ノードを 1 枚のカードで表示。インデントと左ボーダーで階層を表現。
- **テンプレート**: 右パネルにテンプレートカードを表示。クリックで選択ノードの子（またはルート直下）に追加。ドラッグで任意の位置に追加。
- **編集**: タイトル編集、インデント / アウトデント、ドラッグで移動、削除。子の開閉。
- **GROWI から読み込み**: 現在パスの直下の子ページを取得し、ツリーとして表示。
- **保存**: ツリーをブラウザの localStorage に保存（ルートパスごと）。

## 必要な環境

- GROWI（オンプレ / GROWI.cloud）
- ブラウザから `/_api/v3/pages/list` が同一オリジンで利用できること

## インストール

1. このリポジトリを GROWI のプラグインとしてインストール（管理画面の「プラグイン」でリポジトリ URL を指定）
2. プラグインを有効化
3. サイドバーに「📋 ツリー構築」が表示されるのでクリックしてパネルを開く

## 開発

```bash
npm install
npm run build
```

## 計画

- Phase 1–2: カード表示・選択・編集・ドラッグ（実装済み）
- Phase 3: テンプレートパネル（実装済み）
- Phase 4: GROWI へのツリー保存（ページ API 連携は要検討、現状はローカル保存）
- Phase 5: コネクタ・キーボード・アクセシビリティ
